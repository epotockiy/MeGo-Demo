# MeGo-Demo
Create html page based on the providede design.

## Requirements
* Use gulp/grunt and sass
* Pixel perfect
* Create mobile version
* Use following methods to create columns
  * float
  * inline-block
  * table (display property, **NOT <code>&lt;table&gt;</code>**)
  * flex
